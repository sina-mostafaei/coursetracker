# Student Management System - C++



This is a simple C++ program designed to manage student information and their courses. It allows you to add students, input their course details (name, credit, score), find a specific student by name and last name, and identify the top-performing student based on average grade.

**Features:**

*   Add Student Information:  Input student names and last names.
*   Manage Courses: Add courses for each student with name, credit, and score.
*   Search Students: Find a student by name and last name.
*   Find Top Student: Identify the student with the highest average grade.
*   Display Student Details: Print student information and their course details.
